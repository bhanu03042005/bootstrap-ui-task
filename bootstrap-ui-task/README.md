# Bootstrap 5 UI Exploration Task

## Project Overview

This project was developed as part of the Bootstrap 5 UI Exploration internship task. The goal was to understand Bootstrap components, responsive layouts, and page composition by creating a modern multi-page website.

The project includes:

* Home Page
* About Page
* Contact Page

The website was designed using Bootstrap 5, HTML5, and CSS3.

---

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* VS Code
* GitHub
* GitHub Pages

---

## Steps Followed

1. Explored Bootstrap 5 documentation and examples.
2. Created the project structure and connected Bootstrap CDN.
3. Built a responsive navigation bar.
4. Designed hero sections, cards, forms, and footer components.
5. Created responsive layouts using Bootstrap grid system.
6. Added custom CSS styling and hover effects.
7. Tested responsiveness on desktop, tablet, and mobile devices.
8. Uploaded the project to GitHub.
9. Deployed the project using GitHub Pages.

---

## Challenges Faced

* Understanding responsive layouts
* Maintaining equal spacing between sections
* Designing mobile-friendly pages

These challenges were solved by using Bootstrap utility classes and testing layouts using browser developer tools.
